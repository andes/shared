# Shared

Shared es un conjunto de código, librerías y servicios para el proyecto ANDES.


## Instalación
```
npm install @andes/shared --save
```

## Demo
Ejecutar la aplicación con `ng serve` y navegar hasta `http://localhost:4200` 
```
ng serve
```
